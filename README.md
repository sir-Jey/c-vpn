# Simple VPN (C + libsodium)

Простой зашифрованный VPN туннель «точка-точка» на языке C для macOS (utun).  
Проект создан для понимания, как на самом низком уровне работает VPN.

---

## О проекте

Это учебный/экспериментальный VPN, который:
- Использует UDP как транспорт
- Шифрует весь трафик с помощью **XChaCha20-Poly1305** (libsodium)
- Работает через виртуальный интерфейс **utun** (macOS)
- Поддерживает несколько клиентов на сервере

**Цель проекта** — максимально понятное демонстрационное решение, а не production-ready VPN.

---

## Как это работает (простыми словами)

### Общая схема

```
[Приложение] → TUN-интерфейс (utunX) → Моя программа → UDP (зашифровано) → Интернет → Сервер → TUN-интерфейс → Приложение
```

Ваша программа выступает **прокладкой** между виртуальным интерфейсом и реальной сетью.

## Установка и запуск

### 1. Подготовка

```bash
brew install libsodium
make
```

### 2. Создание ключа (один раз)

```bash
dd if=/dev/urandom of=vpn.key bs=32 count=1
chmod 600 vpn.key
```

### 3. Запуск

**Сервер:**
```bash
sudo ./vpn_server
```

**Клиент:**
```bash
sudo ./vpn_client
```

## Важные технические моменты

### Почему UDP, а не TCP?

TCP-over-TCP плохо работает. При потере пакетов начинается "лавина" повторных передач. Поэтому VPNы почти всегда используют UDP.

### TUN vs TAP

- **TUN** — работает на уровне IP (3 слой). Именно он используется в этом проекте.
- **TAP** — работает на уровне Ethernet (2 слой), позволяет передавать broadcast и ARP.

### Инкапсуляция пакетов

```
Оригинальный IP-пакет
        ↓
XChaCha20-Poly1305 (шифрование)
        ↓
Nonce (24 байта) + Зашифрованные данные
        ↓
UDP датаграмма
```
